# Z03 L4 全自动级联 E2E 验收清单

## 1. 功能状态走查

- [ ] L4 未开放（feature flag 关闭时，L4 tab 显示提示）
- [ ] 试点申请审批中
- [ ] 已审批配置中（触发条件、频率上限、回滚策略、通知）
- [ ] 全自动执行成功（DWD 数据刷新 → DWS 自动发布 → ADS 自动发布）
- [ ] WARN 自动继续（中风险告警但自动执行）
- [ ] REVIEW_REQUIRED 待确认（中风险阻断，生成草稿等待确认）
- [ ] APPROVAL_REQUIRED 待审批（高风险阻断，等待审批）
- [ ] FAILED 保留旧版本（技术失败时旧版本仍可用）
- [ ] 频率上限退化草稿（24h 内达到上限，退化为 draft_mode）
- [ ] 紧急停止中（全局紧急停止后新事件不触发执行）
- [ ] 一键回滚成功（基于 l4_publish_batches 批次表回滚）
- [ ] 审计时间线完整（trace_id 贯穿全链路）

## 2. 安全与权限

- [ ] 普通用户不能访问 L4 试点管理页（仅管理员可审批/撤销/紧急停止）
- [ ] 普通用户只能查看自己指标的 L4 状态
- [ ] 审计不记录 UCP secret
- [ ] L4 系统规则默认 disabled
- [ ] 仅低风险指标可通过审批（medium/high 被拒绝）

## 3. 回滚演练

- [ ] DWS 已发布、ADS 失败时自动补偿（auto_rollback=true）
- [ ] 最近一次 L4 自动发布可一键回滚
- [ ] 多次发布可逐次回滚
- [ ] 回滚后批次表标记 rollback_status

## 4. 紧急停止演练

- [ ] 管理员点击紧急停止 → DB l4_runtime_controls.emergency_stop=true
- [ ] 新事件到达 → L4CascadeEngine 检查 is_emergency_stopped() → 跳过
- [ ] 全局状态条展示 '紧急停止中'
- [ ] 管理员点击恢复 → DB + 内存恢复正常
- [ ] 审计日志记录紧急停止操作

## 5. 红线检查

- [ ] 不绕过权限、血缘、质量门禁
- [ ] 不在 ODS 层做业务口径加工
- [ ] 不提供任意 SQL/脚本编辑器
- [ ] 不跳过 DWS 直接 ODS→ADS
- [ ] L4 默认全局禁用
- [ ] 仅审批通过的低风险指标可配置全自动规则
- [ ] 首批试点指标数量 ≤ 5
