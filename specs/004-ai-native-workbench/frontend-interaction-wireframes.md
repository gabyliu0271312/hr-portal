# AI + Excel 计算字段前端交互示意图

版本：v0.1  
日期：2026-06-08  
适用范围：AI 基础配置、Excel 公式组件、数据集级计算字段、报表编辑器接入

## 1. 总体入口流

```text
报表列表
  -> 新建/编辑报表
    -> 选择数据来源 = 数据集
      -> 报表设置 / 可选字段
        -> 新建字段
          -> 公式字段编辑抽屉
            -> 自然语言生成公式
            -> 校验/预览
            -> 保存为数据集计算字段
              -> 回到可选字段
                -> 计算字段分组出现新字段
```

注意：

- 单表报表首期不显示“新建字段”。
- 数据集报表才显示“新建字段”。
- 保存后的计算字段属于数据集，不属于单个报表。

## 3. 新建字段抽屉

组件建议：`FormulaFieldEditor.vue`

```text
┌──────────────────────────────────────────────────────────────┐
│ 新建计算字段                                           [×]   │
├──────────────────────────────────────────────────────────────┤
│ 基本信息                                                     │
│ 字段名称  [ 个税金额____________________ ]                    │
│ 字段编码  [ calc_tax_amount_____________ ]                    │
│ 数据类型  [ 数值 v ]       聚合角色 [ 度量 v ]                 │
│ 描述      [ 根据基本工资计算个税_________ ]                    │
├──────────────────────────────────────────────────────────────┤
│ AI 生成公式                                                  │
│ 自然语言描述                                                 │
│ ┌──────────────────────────────────────────────────────────┐ │
│ │ 用基本工资计算个税，如果为空则返回0                       │ │
│ └──────────────────────────────────────────────────────────┘ │
│ [生成公式] [清空]                                            │
├──────────────────────────────────────────────────────────────┤
│ 公式编辑                                                     │
│ ┌──────────────────────────────────────────────────────────┐ │
│ │ =IF(ISBLANK(FIELD("salary.基本工资")),0,                 │ │
│ │     CALC_TAX(FIELD("salary.基本工资")))                  │ │
│ └──────────────────────────────────────────────────────────┘ │
│ [插入字段] [插入函数] [校验公式]                              │
├──────────────────────────────────────────────────────────────┤
│ 校验结果                                                     │
│ ✓ 公式合法                                                   │
│ ✓ 依赖字段：工资.基本工资                                    │
│ ⚠ 结果继承敏感属性，因为依赖“基本工资”                        │
├──────────────────────────────────────────────────────────────┤
│ 预览                                                         │
│ ┌──────────────┬──────────────┬──────────────┐              │
│ │ 基本工资      │ 个税金额      │ 状态          │              │
│ ├──────────────┼──────────────┼──────────────┤              │
│ │ 15000        │ 790          │ 正常          │              │
│ │              │ 0            │ 空值兜底      │              │
│ └──────────────┴──────────────┴──────────────┘              │
├──────────────────────────────────────────────────────────────┤
│                                      [取消] [保存并加入报表]  │
└──────────────────────────────────────────────────────────────┘
```

交互规则：

- “生成公式”调用 LLM 草稿接口。
- LLM 结果只回填草稿，不自动保存。
- “校验公式”调用后端校验接口。
- 只有校验通过才允许保存。
- “保存并加入报表”保存到数据集，并把字段加入当前报表已选字段。

## 8. 保存后回到报表编辑器

```text
保存成功
  -> Toast: 已保存计算字段“个税金额”
  -> 关闭抽屉
  -> 刷新 allColumns
  -> 计算字段分组新增字段
  -> 若用户点击“保存并加入报表”，同步加入已选字段
```

## 9. 同数据集复用流程

```text
报表 A
  -> 数据集：月度薪酬数据集
  -> 新建计算字段：个税金额
  -> 保存

报表 B
  -> 数据集：月度薪酬数据集
  -> 打开报表编辑器
  -> 可选字段自动出现：计算字段 / 个税金额
```

不需要复制字段，也不需要在报表 B 重新创建。

## 11. 函数库管理页面

建议位置：

```text
系统设置 -> 参数配置 -> 函数库管理
```

定位：

```text
函数库管理是平台公共能力，不属于 AI 模块。
AI 公式助手、报表计算字段、补偿金工具、证明变量计算都可以引用函数库。
```

```text
┌──────────────────────────────────────────────────────────────┐
│ 函数库管理                                      [+ 新增函数]   │
├──────────────────────────────────────────────────────────────┤
│ 函数名        类型              返回值       状态      操作     │
│ CALC_TAX      system_builtin    number       启用      编辑     │
│ ROUND_MONEY   expression        number       启用      编辑     │
│ GET_SALARY    data_action       number       停用      查看     │
└──────────────────────────────────────────────────────────────┘
```

交互规则：

- 系统函数只能查看说明和启停，不能编辑实现。
- 表达式型函数可以维护参数、返回值和公式体。
- `data_action` 类型函数首期只能登记和查看，不允许在计算字段中执行。
- 函数库提供给公式组件的“插入函数”弹窗使用。

## 15. 关键确认点

- “新建字段”是数据集级能力，不是报表级能力。
- LLM 生成的是公式草稿，不自动保存。
- 公式保存前必须后端校验。
- 保存成功后，字段立即可加入当前报表。
- 其他报表只要基于同一数据集，就能复用该字段。
- 自定义函数首期不支持代码执行。
- 自定义函数归函数库管理，不放在 AI 基础配置。
- AI 调用日志归统一日志管理，不放在 AI 基础配置。
- 敏感依赖自动继承到计算字段。
