既有模块扩展 UI 蓝图接入系统数据资产详情事件中心与流程编排滴滴生命周期

不是重建产品：在现有数据仓库和数据连接页面上扩展

本蓝图冻结模块归属与页面关系。它保留已有路由、列表、详情、流程设计器和运行中心，只在正确页面增加能力入口。

数据资产属于数据仓库。
“来源于开放 / 配置入仓来源”继续是数据仓库的独立能力；Offer 富化是该资产详情的附加规则，不会在数据连接中新建“数据资产”。
事件中心不是流程编排。
事件中心负责接收、验签、标准化、去重、追踪与触发；流程编排定义执行步骤。事件触发器只负责把事件绑定到已发布的流程。
