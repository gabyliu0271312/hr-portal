标准 SaaS 能力包与实例资源：统一入口、不同配置路径

静态技术原型，仅用于冻结信息架构、字段、按钮、状态和页面跳转；不代表最终视觉稿。普通用户不接触 API URL、Adapter Code、Token 或 Scope 原始编码。

标准 SaaS：系统 → 凭证 → 业务能力 → 编排实例资源：系统 → 凭证 → 具体对象资源 → 编排写操作：必须审批、幂等、试运行和审计
