## Design Context

### Users
HR Portal 面向多角色混合用户，包括 HRBP、薪酬绩效、系统管理员、IT/接口维护人员以及部分业务协作角色。用户主要在企业内网环境中完成权限管理、数据源配置、报表中台、接口推送、流程与数据校验等工作，关注准确性、可追溯性和低出错率。

### Brand Personality
整体气质应专业、稳健、可信。界面需要让业务用户有信心操作敏感数据和系统配置，也要让 IT/管理员能快速定位配置项、连接信息和执行状态。

### Aesthetic Direction
沿用现有 Vue 3 + Element Plus 企业后台风格、浅色主题、蓝色主色和既有设计 token。避免强烈装饰、炫技动效或与当前系统割裂的视觉语言；局部交互可以更清晰、更顺手，但不改变整体产品气质。

### Design Principles
1. 配置类界面优先清晰、可确认、低误操作，关键操作必须有明确反馈。
2. 保持现有 Element Plus 和设计 token 体系，新增组件融入当前后台风格。
3. 对敏感信息采取渐进披露，默认隐藏，显式操作后展示或重置。
4. 文案短而准确，面向业务用户解释结果，面向管理员保留必要技术细节。
5. 小范围增强优先，不为单个控件引入重型视觉或复杂交互。
